Breeze.css Documentation

A minimal utility-first CSS framework inspired by Tailwind, but simpler and lighter.

Installation

Include the generated CSS file in your project:

<link rel="stylesheet" href="dist/breezecss.css" />

Basic Usage

Use utility classes directly in your HTML:

<div class="p-4 bg-blue-100 text-blue-900 rounded">
    Hello Breeze!
    </div>
Hello Breeze!

Core Utilities

  • Spacing: .p-1 to .p-6, .m-1 to .m-6
  • Flexbox & Grid: .flex, .items-center, .justify-between, .grid, .grid-cols-2, .gap-4
  • Display: .block, .inline-block, .hidden
  • Text: .text-center, .text-left, .font-bold, .text-sm to .text-3xl
  • Colors: .bg-{color}-{shade}, .text-{color}-{shade}, .border-{color}-{shade}
  • Borders: .border, .border-2, .rounded, .rounded-lg
  • Buttons: .btn, .btn-primary, .btn-secondary, .btn-danger
  • Forms: .input, .select, .textarea
  • Responsive: Prefix with sm:, md:, lg: (e.g. md:text-xl)
  • Dark Mode: Add data-theme="dark" to <html>
  • Animations: .fade-in, .slide-in, .bounce

Dark Mode

Enable dark mode by setting data-theme="dark" on the <html> tag.

<html data-theme="dark">
This would appear in dark mode

Edit the source files in utilities/ and re-run node build.js to generate your own build.
